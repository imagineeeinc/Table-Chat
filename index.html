DEV Note

DEV Note is a powerful, small, minimlistic Web bassed Note Taking app. Its also PWA(Progresive Web app), So you can install it on to your device so that you can run it offline.

____________________

WRITE

You need a note pad to write notes, Well thats what DEV Note is built for for taking notes on the go, like grocery shopping.

____________________

SAVE

You need to Save the note for later, Well press the Save button and it is saved to your browser.

____________________

Clear

Maybe You need a fresh clean slate to work with, click the Clear button.

____________________

Save as

Maybe You need to save to your local driv, well just click the Svae as button and it will download a text file.
